Code

Below is the basic code required to use the lightbox. Having fade as part of the class will make use of Twitter Bootstrap Transitions (if you have them included) when displaying and hiding the lightbox.

For this to work you do need jQuery, like the rest of the Twitter Bootstrap plugins, and then all it takes is my the JavaScript and CSS files in my git repository.

<a data-toggle="lightbox" href="#demoLightbox">Open Lightbox</a>
<div class="lightbox fade" id="demoLightbox" style="display: none;">
	<div class='lightbox-content'>
		<img src="image.png">
	</div>
</div>

Bonus!

You can make use of the Twitter Bootstrap Carousel plugin inside the lightbox!
